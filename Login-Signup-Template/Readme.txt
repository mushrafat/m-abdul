This is a responsive login template built using a mobile-first approach.
Style.css file contains the general style and mobile screen style.
Flex.css contains the style for larger screen sizes.

To reduce the number of css files, you can copy and paste the code in
flex.css at the end of style.css then remove the link to flex.css inside
index.html.You can change and cusomize the template to your need.

Features: Built with semantic HTML5 & CSS3. Simple to implement and easy to use. 
Supported Browser: Latest Versions of Firefox, Chrome, Opera, Torch
